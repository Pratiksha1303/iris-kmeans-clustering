This project performs K-Means clustering on the Iris dataset.

Dataset:
Iris flower dataset from sklearn.

Features Used:
Petal Length and Petal Width.

Process:
1. Load iris dataset
2. Select petal features
3. Apply scaling
4. Use elbow method to find optimal K
5. Apply K-Means clustering
6. Visualize clusters

Libraries used:
pandas
matplotlib
scikit-learn
